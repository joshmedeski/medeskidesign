Tools

What I use to do it

Content Management

Wordpress is the most powerful content management system out there.

Collaboration

Collaboration is all about communication. It's important to have a good system that allows both parties to easily talk to each other, avoid missed deadlines, or make assumptions. I'm a very flexible with the systems I use. I want to use what's best for you. This inludes, Asana, Trello, Google Drive, Dropbox, and Evernote.

Coding Framework

There is no reason for every project to be written from the group up. I have learned from all of my past projects and continue to learn with every new project. Zurb's Foundation is a front-end framework that offers great code built by thousands of people. And my framework, Rootbeer offers Wordpress integration with Foundation.

Version Control

Version control is a system that records all of the changes made during a project. It allows multiple users to work on the same project without overriding eachothers work. It also allows you to go back and restore to previous changes, in case anthing goes wrong. I use a system called Git. You can see some of my Git projects on Github if it fancies you.
